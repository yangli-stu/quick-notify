关于 Quick-Notify

开源免费 • 企业级 • 生产验证

企业级实时消息推送

Quick-Notify 是一个基于 Spring Boot 3 + WebSocket STOMP + Redis 的企业级实时消息推送系统。 经过万级日活生产环境验证，稳定可靠，适用于电商订单通知、社交消息推送、在线客服系统、实时监控系统等多种场景。

支持集群部署、消息 ACK 确认、多设备同步、消息持久化等企业级特性，让开发者专注于业务逻辑，快速实现实时消息推送功能。

核心特性

开箱即用，企业级特性助您快速实现实时消息推送

Starter 模式

一行依赖即可接入，零配置启用，Spring Boot 自动装配，开箱即用。

ACK 确认机制

自动重试机制，消息可靠送达，网络抖动也能保证消息不丢失。

集群支持

基于 Redis Pub/Sub，多节点水平扩展，支撑高并发场景。

多设备同步

同一用户多设备独立确认机制，消息不遗漏，体验一致。

消息持久化

支持历史消息存储与查询，消息可追溯，审计无忧。

安全认证

Token 鉴权，连接数限制，安全加固，放心使用。
